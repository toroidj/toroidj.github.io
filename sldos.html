↑K-Launcher Version 1.30 動作画面
   KL130.LZH ( 139 Kbytes, 2016-9-17, PC-98x1,PC/AT,FM-R,TOWNS,J3100,PC-98LT/HA,PC-88VA,ANSI)
    (source: KL130S.LZH ,164 Kbytes, 2016-9-17)

色々なMS-DOSマシンで動作するファイル管理ツールです。
DOS or Windows95/98/Me の FAT12/16/32上でディレクトリの書き換えができます。
また、LFNによるファイル操作ができます。

●変更点
1.29→1.30
・一部のDOSエミュレータ(DOSBox)でDir読込時に異常終了することに対する対策をした
・64Kクラスタに対応

●その他
Version 1.00 以前の履歴：KLUPDATE.LZH
補足情報
↑K-Launcher 20000カスタマイズ差分 Version 1.30
   KL20K130.LZH ( 9 Kbytes, 2016-9-17, PC-98x1,PC/AT)

K-Launcher のカスタマイズ領域を通常の 6000/12000 から 20000 に拡張する差分です。
PC-98x1 版(KL98.EXE)、IBM-PC 版(KLPC.EXE)、DOS/V 版(KLV.EXE)が対象です。
↑K-Launcher Version 1.15+2(Xmas edition)
   KL115P2S.LZH ( 57 Kbytes, 1998-12-12, PC-98x1,PC/AT,PC-88VA)

試験公開版です。 Version 1.15からの差し替え分のみとなってます。

ファイル管理ツール K-Launcher の限定バージョンです。通常バージョンにおまけ機能を追加しています。
詳細は通常バージョン内のマニュアルを参照して下さい。

再配布は不可です。

↑CT Version 1.3
   ct13.zip ( 11 Kbytes, 2013-2-23, MS-DOS2.1以上／Windows95～Me/NT3.51～8)

あるファイルのタイムスタンプの内容を別のファイルのタイムスタンプにコピー
します。また、MS-DOS 上では DOS ソフトとして動作し、Windows 95/NT 等では
32bit コンソールアプリケーションとしてネイティブ動作をします。

●変更点
1.2→1.3
・(WIN)エラー表示を調整
